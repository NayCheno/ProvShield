# Evaluation Starter

This folder defines the evaluation structure for ProvShield.

## Main outputs

- `benchmark_matrix.csv`: experiment grid.
- `attack_suite_plan.yaml`: attack categories and success conditions.
- `benign_tasks.yaml`: benign workload.
- `metrics.md`: metric definitions.
- `ablation_plan.md`: ablation experiments.
- `analysis_template.md`: result write-up template.
